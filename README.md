# Twitter Clone

This is a Twitter clone built from scratch using the MERN stack (MongoDB, Express, React, Node.js). Itâ€™s a comprehensive project where I practiced backend development and integrated various features typically found in social media platforms.

## Features

- **User Authentication**: Register and login users with secure authentication.
- **Tweets**: Post, like, and comment on tweets.
- **User Profiles**: View and edit user profiles.
- **Follow System**: Follow and unfollow other users.
- **Messages**: Fetch all followed user messages section.
- **Delete Posts**: Users can delete their own posts.
- **Explore**: Discover more content and users.

## Usage

1. **Register an account**.
2. **Login with your credentials**.
3. **Explore the features**:
   - Post tweets
   - Like and comment on tweets
   - Follow other users
   - Send and receive messages
   - View and edit profiles
